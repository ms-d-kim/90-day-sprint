## Setup

1. Create and activate virtual environment:
```bash
   python3 -m venv .venv
   source .venv/bin/activate
```

2. Install dependencies:
```bash
   pip install -r requirements.txt
```

## Run
```bash
python src/main.py
```

## What it does

Fetches a sample todo item from [JSONPlaceholder](https://jsonplaceholder.typicode.com/) and prints it.
